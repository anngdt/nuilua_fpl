NLFPL Dashboard

A clean and interactive Fantasy Premier League dashboard built for private mini-leagues.

Features:
- Classic & H2H League Table
- Season Stats
- Rank Journey Visualizations
- Scores vs Scores Against Analytics
- Hit Points vs Rank Scatter
- Head-to-Head Matrix
- Real-time FPL API integration
- Manager Profile
  
Tech Stack:
- Vanilla JavaScript
- HTML5 Canvas
- Netlify Functions
- Fantasy Premier League API

Link: nuiluafpl.netlify.app

Local Development: 
npm install
netlify dev

Notes

This project uses unofficial Fantasy Premier League API endpoints and is intended for personal/private league use.
